1. Einführung: Programmiersprachen und Java

Programmiersprachen dienen dazu, Algorithmen und Datenstrukturen in einer für den Computer verständlichen Form zu formulieren. Java ist eine der weit verbreiteten höheren Programmiersprachen der 3. Generation. Es ermöglicht die Plattformunabhängigkeit, indem es den Quellcode in plattformunabhängigen Bytecode kompiliert, der auf verschiedenen Betriebssystemen ausgeführt werden kann.

2. Java: Geschichte und Eigenschaften

Java wurde 1995 von Sun Microsystems eingeführt. Es war ursprünglich für eingebettete Systeme wie Steuerungselektronik in Haushaltsgeräten gedacht. Aufgrund seiner Plattformunabhängigkeit und des objektorientierten Designs entwickelte es sich schnell zur bevorzugten Sprache für Internetanwendungen.

Java Plattformunabhängigkeit

Java-Programme werden in Bytecode kompiliert, der in jeder Java Virtual Machine (JVM) ausgeführt werden kann. Diese JVMs sind für verschiedene Betriebssysteme verfügbar, was die Ausführung des Codes auf unterschiedlichen Plattformen ermöglicht.

3. Vom Ablaufplan zum Java-Programm

Die Entwicklung eines Java-Programms durchläuft mehrere Schritte:

  1. Präzisierung der Eingaben und Ausgaben: Was sind die Eingangsdaten und welche Ergebnisse werden erwartet?
  2. Entwicklung einer Lösungsidee: Definieren der logischen Schritte zur Lösung des Problems, meist mit Hilfe eines Programmablaufplans (PAP).
  3. Implementierung: Schreiben des Programms in Java.
  4. Testen und Debugging: Überprüfen des Programms auf Fehler und Korrekturen vornehmen.

4. Variablen und Datentypen

Variablen sind Platzhalter, die Werte speichern, und Datentypen definieren, welche Art von Daten in diesen Variablen gespeichert werden können. Java hat verschiedene primitive Datentypen wie int (Ganzzahlen), boolean (Wahr/Falsch) und double (Fließkommazahlen), sowie komplexe Datentypen wie String.

Beispiel: Variablendeklaration in Java

int zahl = 10;
double pi = 3.1415;
String name = "Java";

5. Funktionen und Methoden in Java

Funktionen (in Java oft als Methoden bezeichnet) sind wichtige Bausteine in Java-Programmen. Sie führen spezifische Aufgaben aus und ermöglichen die Strukturierung des Codes. Eine Funktion kann Parameter annehmen, um mit unterschiedlichen Werten zu arbeiten, und einen Rückgabewert liefern.

Methoden in Java

Die grundlegende Syntax einer Methode in Java besteht aus folgenden Teilen:

  • Zugriffsmodifikator (z.B. public, private) legt fest, ob die Methode aus anderen Klassen aufgerufen werden kann.
  • Rückgabewert (z.B. int, boolean, void) gibt den Datentyp des Rückgabewerts der Methode an.
  • Methodenname ist der Name, mit dem die Methode aufgerufen wird.
  • Parameterliste in runden Klammern nimmt die Eingabewerte an, mit denen die Methode arbeitet.

Beispiel: Eine Methode in Java

public int addiere(int a, int b) {
return a + b;
}

In diesem Beispiel ist public der Zugriffsmodifikator, int der Rückgabewert, und addiere ist der Methodenname. Die Methode nimmt zwei Parameter a und b und gibt deren Summe zurück.

6. Implementierung des größten gemeinsamen Teilers (ggT) in Java

Ein klassisches Algorithmusproblem ist die Berechnung des größten gemeinsamen Teilers (ggT) zweier Zahlen. In Java kann dies auf zwei Arten implementiert werden: iterativ und rekursiv.

Beispiel: Iterative ggT-Berechnung in Java

public class GGT {
public static void main(String[] args) {
int a = 36;
int b = 8;
while (a != b) {
if (a > b) {
    a = a - b;
} else {
    b = b - a;
}
}
System.out.println("Der ggT ist: " + a);
}
}

Hier wird eine While-Schleife verwendet, um die Berechnung durch wiederholtes Subtrahieren der kleineren Zahl von der größeren durchzuführen, bis beide Zahlen gleich sind.

Beispiel: Rekursive ggT-Berechnung in Java

public class GGT {
public static int ggT(int a, int b) {
if (a == b) {
return a;
} else if (a > b) {
return ggT(a - b, b);
} else {
return ggT(a, b - a);
}
}

public static void main(String[] args) {
System.out.println("Der ggT ist: " + ggT(36, 8));
}
}

Dies ist eine rekursive Lösung, bei der sich die Methode selbst aufruft, bis eine Abbruchbedingung erreicht ist (d.h. a == b).

7. Kontrollstrukturen in Java

Java bietet verschiedene Kontrollstrukturen zur Steuerung des Programmablaufs, z.B. If-Anweisungen, While-Schleifen und For-Schleifen. Diese ermöglichen es, bestimmte Teile des Codes basierend auf Bedingungen auszuführen oder zu wiederholen.

Beispiel: If-Anweisung

int x = 10;
if (x > 5) {
System.out.println("x ist größer als 5");
} else {
System.out.println("x ist kleiner oder gleich 5");
}

8. Konsolenausgabe in Java
