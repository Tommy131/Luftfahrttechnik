4. Kurzfragen (WS2223, ca. 4 Punkte)

In Java kann man Methodennparameter übergeben. Für diese Parameter gilt entweder die „call by value“ oder die „call by reference“ Übergabe.

1a) Erklären Sie verständlich was der Unterschied zwischen den beiden Möglichkeiten ist.

Value: Kopie der Variablen als Parameter übergeben, Änderungen am Parameter haben keine Auswirkung auf die ursprüngliche Variable.
Ref: Es wird ein Zeiger (Referenz) auf die gleiche Adresse der Variable übergeben. Änderungen am Parameter führen auch zur Änderung des ursprünglichen Objekts.

1b) Geben Sie zwei Beispiele von Datentypen für die „call by value“ und zwei für „call by reference“ benutzt wird.

Value: Primitive Datentypen (z.B. int, double, char, boolean).
Ref: Alle Objekte (selbst definierte Klassen, Arrays, String).

Zusammenfassung der vier Algorithmen

1. Harmonic Number Berechnung

Problem: Berechnen und geben Sie die n-te harmonische Zahl zurück, d.h.

H_n = 1 + 1/2 + 1/3 + ... + 1/n

Algorithmusschritte:

  • Initialisieren Sie die Teilsumme auf 0 und setzen Sie den Zähler k = 1.
  • Schleife von k = 1 bis n, fügen Sie in jeder Iteration 1/k zur Teilsumme hinzu.
  • Geben Sie die berechnete Teilsumme zurück.

Java Code Implementierung:

public double berechne(int n) {
    double s = 0;
    for (int k = 1; k <= n; k++) {
        s += 1.0 / k;
    }
    return s;
}

2. Teiler Berechnung

Problem: Gegeben eine ganze Zahl n, ermitteln und geben Sie alle Teiler aus.

Algorithmusschritte:

  • Initialisieren Sie den Wert t = 1, um von 1 bis n/2 zu iterieren.
  • Verwenden Sie eine while-Schleife, um zu prüfen, ob jeder Wert ein Teiler von n ist.
  • Wenn ein Teiler gefunden wird, geben Sie ihn aus.
  • Die Schleife läuft weiter, bis n/2 erreicht ist.

Java Code Implementierung:

public void teile(int n) {
    for (int t = 1; t <= n / 2; t++) {
        if (n % t == 0) {
            System.out.println(t);
        }
    }
    System.out.println(n);  // n ist auch ein Teiler von n
}

3. Quersumme Berechnung

Problem: Gegeben eine Zahl z, berechnen Sie die Quersumme (Summe der Ziffern).

Algorithmusschritte:

  • Initialisieren Sie die Quersumme s = 0.
  • Extrahieren Sie jede Ziffer und addieren Sie sie zur Quersumme.
  • Verwenden Sie die Modulo-Operation (z % 10) um die Ziffer zu extrahieren und die Division (z / 10) um die Ziffer zu entfernen.
  • Wiederholen Sie dies, bis z gleich 0 ist, und geben Sie die Quersumme s zurück.

Java Code Implementierung:

public int berechneQuersumme(int z) {
    int s = 0;
    while (z > 0) {
        s += z % 10;
        z /= 10;
    }
    return s;
}

4. Näherung des natürlichen Logarithmus (ln(2))

Problem: Berechnen Sie den natürlichen Logarithmus ln(2) mithilfe einer Reihe.

ln(2) = 1 - 1/2 + 1/3 - 1/4 + ...

Algorithmusschritte:

  • Initialisieren Sie die Teilsumme s = 0 und den Wechsel der Vorzeichen z = 1.
  • Verwenden Sie eine while-Schleife, um jeden Term zu berechnen und zur Teilsumme hinzuzufügen.
  • Ändern Sie das Vorzeichen jedes Mal, indem Sie z = -z setzen.
  • Schließen Sie die Berechnung ab und geben Sie die Teilsumme s zurück.

Java Code Implementierung:

public double berechneLn2(int n) {
    double s = 0;
    double z = 1;
    for (int k = 1; k <= n; k++) {
        s += z / k;
        z = -z;  // Vorzeichen wechseln
    }
    return s;
}
