1. Algorithmen: Definition und Eigenschaften

Ein Algorithmus ist eine Schritt-für-Schritt-Anleitung zur Lösung eines Problems. Er hat folgende Eigenschaften:

  • Endlichkeit: Der Algorithmus muss nach einer endlichen Anzahl von Schritten enden.
  • Eindeutigkeit: Jeder Schritt muss klar und eindeutig beschrieben sein.
  • Determiniertheit: Bei gleichen Eingaben muss der Algorithmus immer die gleichen Ergebnisse liefern.
  • Effizienz: Der Algorithmus sollte so wenig Ressourcen (Zeit, Speicher) wie möglich verbrauchen.
  • Allgemeinheit: Ein Algorithmus sollte für eine Klasse von Problemen anwendbar sein, nicht nur für eine spezifische Aufgabe.

2. Berechenbarkeit von Algorithmen

Nicht alle Algorithmen sind berechenbar. Ein Beispiel für eine nicht berechenbare Funktion ist die Berechnung aller reellen Zahlen im Intervall [0,1]. Ein solches Problem kann auf realen Rechnern nicht gelöst werden.

3. Vom Algorithmus zum Programm

Der Übergang vom Algorithmus zum Programm umfasst mehrere Schritte:

  1. Definieren der Eingabe und Ausgabe.
  2. Entwicklung einer Lösungsidee.
  3. Erstellen eines Programmablaufplans (PAP) oder in Pseudocode.
  4. Umsetzen des Algorithmus in eine Programmiersprache.
  5. Testen und Debugging.

4. Beispiel: Berechnung des größten gemeinsamen Teilers (ggT)

Der größte gemeinsame Teiler (ggT) zweier Zahlen a und b ist die größte Zahl, durch die sowohl a als auch b teilbar sind.

Algorithmus nach Euklid (klassisch, iterativ)

Durch wiederholtes Subtrahieren der kleineren Zahl von der größeren, bis beide Zahlen gleich sind, wird der ggT berechnet.

while a != b:
if a > b:
a = a - b
else:
b = b - a
return a

Moderner euklidischer Algorithmus (mit Modulo)

Ein effizienterer Algorithmus verwendet die Division mit Rest (Modulo):

def ggT(a, b):
while b != 0:
a, b = b, a % b
return a

# Beispiel:
print(ggT(36, 8))  # Ausgabe: 4

5. Variablen und Datentypen

Eine Variable ist ein Platzhalter für Werte, die der Computer während der Programmausführung speichert und manipuliert. Ein Datentyp gibt an, welche Art von Daten in einer Variable gespeichert werden können, z.B. int für ganze Zahlen oder boolean für wahr/falsch.

6. Iteration und Rekursion

Wiederholungen in einem Algorithmus können iterativ oder rekursiv sein:

  • Iteration: Die Wiederholung wird durch Schleifen wie for oder while realisiert.
  • Rekursion: Eine Funktion ruft sich selbst auf, bis eine Abbruchbedingung erfüllt ist.

Beispiel: Rekursive Berechnung der Fakultät

def fakultaet(n):
if n == 0:
return 1
else:
return n * fakultaet(n - 1)

# Beispiel:
print(fakultaet(5))  # Ausgabe: 120

7. Schaltjahrberechnung

Ein Schaltjahr folgt den folgenden Regeln:

  • Wenn das Jahr durch 4 teilbar ist, ist es ein Schaltjahr.
  • Ausnahme: Jahre, die durch 100 teilbar sind, sind keine Schaltjahre.
  • Jahre, die sowohl durch 100 als auch durch 400 teilbar sind, sind wiederum Schaltjahre.

Algorithmus zur Schaltjahrberechnung

def ist_schaltjahr(jahr):
if jahr % 400 == 0:
return True
elif jahr % 100 == 0:
return False
elif jahr % 4 == 0:
return True
else:
return False

# Beispiel:
print(ist_schaltjahr(2000))  # Ausgabe: True
print(ist_schaltjahr(1900))  # Ausgabe: False
